Combining [clsx](https://www.npmjs.com/package/clsx) or [classnames](https://www.npmjs.com/package/classnames) with [tailwind-merge](https://www.npmjs.com/package/tailwind-merge) allows us to conditionally join tailwind classes in `classNames` without style conflicts.


```tsx showLineNumbers title="lib/utils.ts"
import { clsx, type ClassValue } from "clsx";
import { twMerge } from "tailwind-merge";

export function cn(...args: ClassValue[]) {
  return twMerge(clsx(args));
}
```
