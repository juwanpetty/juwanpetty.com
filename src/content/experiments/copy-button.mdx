title: Copy Button
description: A copy button component using Motion that can be used to copy text to the clipboard
published: 2026-03-27
---

Small interface details can make interactions feel intentional instead of mechanical. This copy button is a simple example of that idea. 

When clicked, it briefly swaps its icon from a copy symbol to a checkmark, giving immediate visual confirmation that the action succeeded. After a short delay, it resets itself so the button is ready to be used again.
