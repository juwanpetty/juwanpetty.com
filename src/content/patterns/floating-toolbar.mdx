A floating toolbar built with Shadcn UI's Tooltip and Button components.

This utilizes the `delayDuration` prop of the Tooltip component to set a short delay for the initial tooltip appearance. Subsequent tooltips can then appear with little to no delay, allowing for a quicker tooltip interaction as users move their cursor across the toolbar.
